# Weather Data Visualizer - [ANKIT KUMAR]

## Project Overview
This project analyzes weather data using Python. It includes data cleaning, statistical analysis, visualizations, and insights from a real CSV dataset.

## Dataset
- Source: Kaggle (weatherAUS.csv)
- Description: Daily weather data with temperature, rainfall, and humidity.

## Tools and Libraries
- Python, Pandas, NumPy, Matplotlib
- Jupyter Notebook for execution

## Results
- Visualizations: Daily temp trends, monthly rainfall, humidity scatter, combined plots.
- Insights: Seasonal patterns, anomalies in weather data.

## Files
- `weather_data_visualizer.ipynb`: Main notebook.
- `cleaned_weather_data.csv`: Processed data.
- PNG images: Charts.
- `summary_report.md`: Detailed report.
